XDM Visualization

Prerequisite

1. Ruby gem version 2.6.14 (use "sudo gem install -n /usr/local/bin sass" for sass install)

2. Grunt grunt-cli version 1.3.2 (npm install -g grunt-cli)

Visualization for global XDM objects such as uber profile/experienceevent etc.

1. Under the root of git branch, run ./scripts/globalXdmVisual.sh authorizationToken. Example: ./scripts/globalXdmVisual.sh eyJ4NXUiOiJpbXNfbmExLXN0ZzEta2V5LTEuY2V

2. Go to http://localhost:9001/prod/ to see XDM visualization in D3.

Visualization for a single XDM object from API call

1. Under the root of git branch, run ./scripts/xdmVisual.sh authorizationToken sourceId/xdmType/metaAltId. Example: ./scripts/xdmVisual.sh eyJ4NXUiOiJpbXNfbmExLXN0ZzEta2V5LTEuY2V tenant/schemas/_xdm.context.profile__union

2. Go to http://localhost:9001/prod/ to see XDM visualization in D3.
